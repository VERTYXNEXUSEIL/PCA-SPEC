# PCA Glossary

- **PCA**: Proof-Carrying Autonomy.
- **CAE**: Certified Agentic Execution.
- **PC**: Plan Certificate.
- **PC Digest**: Canonical hash binding plan evidence and constraints.
- **AIR**: Action Intermediate Representation.
- **Constraints Digest**: Hash of canonical constraints payload.
- **Effective Time**: Validity window for execution authorization.
- **TOCTOU**: Time-of-check/time-of-use inconsistency class.
- **BRS**: Behavioral Risk Score.
- **OOD**: Out-of-Distribution condition.
- **Fallback**: Safe non-execution decision when checks fail.
