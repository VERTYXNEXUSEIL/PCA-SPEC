# Release Notes - v1.0.0

## Overview

First public release of the **Proof-Carrying Autonomy (PCA) / Certified Agentic Execution (CAE)** standardization package and reference implementation.

## Highlights

- Published normative PCA specification suite (`PCA-SPEC`, `PCA-RC`, `PCA-TEST`, glossary, FAQ).
- Published canonical JSON Schemas for PC, constraints, Action IR, and evidence capsule.
- Added deterministic test vectors T1-T13 for conformance validation.
- Released Python reference implementation with CLI and automated tests.

## Provenance and Integrity

- Signed release tag `v1.0.0` intended as first-publication anchor.
- Release artifacts include bundle ZIP and `SHA256SUMS` verification file.
- CI workflows provide repeatable checks for tests, lint, release packaging, and provenance hints.
