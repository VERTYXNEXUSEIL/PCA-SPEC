# Release Notes v1.0.0

Release date: `2026-02-17`

## Summary

- First public release of the normative PCA/CAE specification.
- Canonical JSON schemas in `schemas/`.
- Deterministic test vectors T1-T13.
- Python reference implementation and CLI tool.

## Compatibility

- The standard defines Bronze, Silver, and Gold profiles.
- Reason codes from `spec/PCA-RC-v1.md` are treated as stable identifiers.
