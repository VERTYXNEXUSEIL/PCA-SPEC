# Code of Conduct

This project adopts a professional, respectful collaboration model.

## Our Standards
- Be respectful and constructive.
- Focus on technical merit and reproducibility.
- Avoid harassment, intimidation, and discriminatory behavior.

## Enforcement
Project maintainers may remove, edit, or reject comments, commits, code, and issues that violate this Code of Conduct.

## Reporting
Report concerns to project maintainers via repository security/contact channels (use placeholders as needed).
