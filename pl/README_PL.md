# PCA-SPEC (Polski)

To repozytorium udostępnia standard Proof-Carrying Autonomy (PCA) / Certified Agentic Execution (CAE) oraz implementację referencyjną.

## Zawartość

- Polska wersja specyfikacji: [`pl/spec/`](./spec/)
- Tłumaczenia materiałów wdrożeniowych: [`pl/adoption/`](./adoption/)
- Tłumaczenia materiałów certyfikacyjnych: [`pl/cert/`](./cert/)
- Tłumaczenia profili zgodności: [`pl/profiles/`](./profiles/)
- Polskie noty wydania: [`pl/releases/`](./releases/)
- Dokumenty operacyjne i indeks: [`pl/docs/`](./docs/)

## Uwaga o kompatybilności

W tłumaczeniach celowo nie są zmieniane identyfikatory techniczne: reason code IDs (np. `RC_TOCTOU`), etykiety haszowania (np. `STATEv1`), nazwy pól JSON (np. `constraints_digest`) ani nazwy subkomend CLI.
