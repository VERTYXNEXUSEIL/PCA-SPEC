# Program certyfikacji PCA/CAE (PL)

## Cel

Potwierdzenie, że implementacja wykonuje protokół PCA/CAE deterministycznie i audytowalnie.

## Wymagane dowody

- Wyniki `PCA-TEST v1` adekwatne do deklarowanego profilu.
- Artefakty pokazujące stabilność reason codes.
- Dowody poprawnej walidacji `constraints_digest`, `constraints_version_id` i `effective_time`.
