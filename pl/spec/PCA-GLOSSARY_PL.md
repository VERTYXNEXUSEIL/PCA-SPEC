# PCA Glossary (PL)

- **PCA**: Proof-Carrying Autonomy.
- **CAE**: Certified Agentic Execution.
- **PC**: Plan Certificate.
- **PC Digest**: Skrót kanoniczny wiążący dowody planu i constraints.
- **AIR**: Action Intermediate Representation.
- **Constraints Digest**: Skrót kanonicznego ładunku constraints.
- **Effective Time**: Okno ważności autoryzacji wykonania.
- **TOCTOU**: Klasa niespójności time-of-check/time-of-use.
- **BRS**: Behavioral Risk Score.
- **OOD**: Warunek Out-of-Distribution.
- **Fallback**: Bezpieczna decyzja niewykonania, gdy kontrole nie przechodzą.
