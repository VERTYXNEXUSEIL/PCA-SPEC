# CAE-Mark Guidelines

These guidelines describe neutral conformance claims for PCA/CAE profiles.

## Claim format
Implementers SHOULD use the wording:
- "PCA/CAE Bronze conformant"
- "PCA/CAE Silver conformant"
- "PCA/CAE Gold conformant"

## Claim conditions
A claim SHOULD include:
1. Profile level.
2. Tested PCA-TEST vector set.
3. Release tag and artifact checksums.
4. Public conformance report.

## Scope note
This file defines technical communication guidance only and does not assert trademark ownership.
